1. ¿Qué diferencia a Javascript de cualquier otro lenguaje de programación?

▲

JavaScript (abreviado como JS) es un lenguaje de programación versátil que permite que las páginas web sean interactivas y fáciles de usar. Gracias a él, los sitios web tienen botones, ventanas emergentes, formularios de entrada, reacciones a las acciones del usuario y muchos otros elementos. Sin JavaScript, una página sería estática. Con JavaScript, responde a las acciones del usuario.

📌 Aunque originalmente JavaScript fue creado como un lenguaje para navegadores, hoy en día se utiliza no solo para sitios web, sino también para desarrollar diversos tipos de programas: desde aplicaciones web hasta servidores, aplicaciones móviles y videojuegos.

Dónde se utiliza JavaScript

Frontend de aplicaciones web
Con JavaScript, los desarrolladores crean aplicaciones web, que son programas que ejecutamos en el navegador, así como la «interfaz externa» de las aplicaciones web, con la que interactúa el usuario al hacer clic en botones, cuadros, menús.
El código de servicios populares como Google Maps, Google Docs, Netflix, eBay está escrito en JavaScript.
Backend de aplicaciones web
Además de la interfaz que ven los usuarios, las aplicaciones web también tienen una parte interna: el backend. Esta es la parte del servidor de la aplicación, donde generalmente se almacenan las bases de datos. JavaScript tiene la plataforma Node.js, que no es un lenguaje de programación independiente, sino un entorno de ejecución del código JavaScript en el servidor. Con ella, se pueden realizar operaciones o acceder a bases de datos.
Empresas conocidas como PayPal y Walmart utilizan JavaScript junto con Node.js para interactuar con sus servidores.
Aplicaciones móviles
JavaScript también se utiliza para desarrollar aplicaciones para iOS y Android. Para ello, existen frameworks especiales: conjuntos de código listos, que sirven como «estructura» de futuras aplicaciones. Los frameworks de JavaScript como React Native ayudan a adaptar el código a las plataformas móviles. Esto permite crear aplicaciones multiplataforma, que funcionan tanto en iOS como en Android, sin necesidad de contratar desarrolladores para cada una de estas plataformas y crear dos versiones de la aplicación.
Las aplicaciones móviles de Groupon y LinkedIn están escritas en JavaScript.
Aplicaciones de escritorio
El popular entorno de desarrollo VS Code fue escrito en el framework Electron y JavaScript.
Juegos
ambién se pueden escribir juegos simples en 2D/3D con JavaScript. Normalmente, estos son juegos de aventura en el navegador.
Aprendizaje automático
El aprendizaje automático suele requerir otros lenguajes de programación, como Python. Sin embargo, a veces también se utiliza JavaScript. Por ejemplo, cuando se crean aplicaciones web, incluidas aquellas que visualizan datos de modelos de aprendizaje automático. O cuando se escribe una página web con un formulario que envía datos a un servidor para ser procesados por un modelo de aprendizaje automático.
Bases de datos
JavaScript tiene su propia herramienta, Mongoose, que permite trabajar con la popular base de datos MongoDB. También existen bases de datos como Clusterpoint, a las que se puede escribir consultas en JavaScript. Además, existe GraphQL, un lenguaje de consultas estructuradas que permite acceder a bases de datos. Funciona con React y otros frameworks populares de JavaScript.
Internet de las cosas (IoT)
JavaScript también se utiliza en el Internet de las cosas (IoT), que es un sistema para intercambiar información entre dispositivos conectados a una red. Ejemplos de estos dispositivos son bien conocidos: relojes inteligentes, rastreadores de actividad, alarmas de incendio. Para ellos, se escriben programas en JavaScript utilizando frameworks como Cylon.js y Johnny-Five.

2. ¿Cuáles son algunos tipos de datos JS?

▲

Cuando escribimos programas en cualquier lenguaje, incluido JavaScript, trabajamos constantemente con datos. Estos datos pueden ser de diferentes tipos: números, texto, objetos, etc. Para que JavaScript entienda correctamente con qué tipo de datos está trabajando, los clasifica en tipos de datos.

📌 Los datos alfanuméricos en JavaScript se definen con el tipo de dato cadena (string). Una cadena puede contener un solo carácter, muchos caracteres o no contener ninguno (una cadena vacía).

Ejemplo: 


    'Hello, World!'
    "Hello, World!"

Cualquier cadena en JavaScript debe estar encerrada entre comillas simples o dobles. No existe una regla estricta sobre cuál tipo de comillas usar. El estándar de codificación de Airbnb recomienda usar comillas simples. Algunas fuentes también sugieren usar comillas simples para el código JavaScript y comillas dobles para el código HTML, pero esto depende de la preferencia personal de cada desarrollador. Lo importante es que el estilo se mantenga consistente en todo el documento.

📌 El tipo de dato número (number) incluye tanto enteros (positivos y negativos) como números reales (decimales, con punto flotante).

Además de los números comunes, este tipo de dato también incluye lo que se conoce como valores numéricos especiales:

  • Infinity representa la infinitud matemática, que es mayor que cualquier número. En programación, es bien conocida la «división por 0». En lenguajes de bajo nivel, esto provoca un fallo en el programa y la necesidad de reiniciarlo. Sin embargo, JavaScript sigue funcionando y devuelve Infinity.
  • -Infinity representa la infinitud matemática negativa, que es menor que cualquier número. Es el resultado de dividir un número negativo entre 0.
  • NaN (not a number) significa error de cálculo, no es un número. Es el resultado de una operación matemática incorrecta o indefinida. Cualquier operación matemática que contenga NaN devuelve NaN. Hay una sola excepción: NaN elevado a la potencia de 0 es igual a 1.

Ejemplo: 


    let age = 25;
    let price = 99.99;

📌 En JavaScript, el tipo de dato numérico solo puede trabajar dentro del rango de números ±(253-1). Fuera de este rango, se producirá un error de precisión, ya que no todos los dígitos caben en una memoria de 64 bits fija. En la mayoría de los casos, el rango seguro de números es suficiente, pero a veces necesitamos trabajar con números realmente gigantes sin ningún límite. En ese caso, se utiliza el tipo de dato bigint. Este tipo permite trabajar con enteros de longitud arbitraria. Para crear un valor de tipo bigint, es necesario agregar una n al final del literal numérico.

Ejemplo: 


    console.log(5465455647852456554235); // 5.465455647852457e+21
    console.log(5465455647852456554235n); // 5465455647852456554235n

Actualmente, el tipo de dato bigint solo es compatible con los navegadores Firefox, Chrome, Edge y Safari.

📌 El tipo de dato booleano (boolean) solo puede tomar dos valores: true (sí, verdadero) o false (no, falso).

Ejemplo: 


    console.log(5 > 2); // true
    console.log(5 < 2); // false

📌 El valor especial null no pertenece a ninguno de los tipos de datos mencionados anteriormente. Forma un tipo de dato independiente. En JavaScript, null no representa un valor numérico cero ni una referencia a un objeto inexistente, como ocurre en algunos otros lenguajes de programación. Es un valor especial que indica: nada, vacío, valor desconocido.

Ejemplo: 


    let user = null; // el usuario no está autorizado

📌 El valor especial undefined también forma un tipo de dato independiente. Indica que algo no está definido, es decir, que no se ha asignado un valor. Este valor puede ser establecido automáticamente por defecto o asignado manualmente (lo cual no se recomienda).

Ejemplo: 


    let item;
    console.log(item); // undefined

📌 Symbol es un tipo de dato primitivo, único e inmutable, que se utiliza para crear identificadores únicos.

Ejemplo: 


    let mySym = Symbol('id');
    console.log(mySym) // id

📌 El tipo de dato objeto (object) es más complejo en comparación con los tipos de datos primitivos mencionados anteriormente. Puede almacenar colecciones de datos o estructuras más complejas. Guarda información en forma de pares clave-valor (key-value).

Ejemplo: 


    let person = {
        name: 'Anne',
        age: 25,
        isStudent: true
    };
    console.log(person.name); // Anne

📌 Un array es una lista ordenada de valores (esencialmente, un objeto con índices numéricos).

Ejemplo: 


    let fruits = ['apple', 'banana', 'pear'];
    console.log(fruits[1]); // banana

📌 Una función (function) es un objeto que se puede ejecutar.

Ejemplo: 


    function greet(name) {
        console.log('Hola, ' + name);
    }
    greet("Anne"); // Hola, Anne

3. ¿Cuáles son las tres funciones de String en JS?

▲

JavaScript nos proporciona una gran cantidad de funciones integradas (métodos) que permiten analizar, modificar o formatear cadenas de texto. En este material, vamos a ver tres funciones (o, más exactamente, métodos) del objeto String, que se utilizan con mayor frecuencia en la práctica real:

📌 El método length muestra cuántos caracteres hay en una cadena. Esto incluye espacios, signos de puntuación, caracteres especiales e incluso emojis.

Ejemplo: 


    let message = '¡Hola, mundo!';
    console.log(message.length); // 13

📌 El método includes() permite verificar si un valor está contenido dentro de una cadena. Devuelve true si lo contiene y false si no.

Ejemplo: 


    let text = '¡JavaScript es genial!';
    console.log(text.includes('genial'));  // true
    console.log(text.includes('Python'));  // false

📌 toLowerCase() convierte todas las letras de una cadena a minúsculas y toUpperCase() convierte todas las letras de una cadena a mayúsculas.

Ejemplo: 


    let name = "Anne";
    console.log(name.toLowerCase()); // anne
    console.log(name.toUpperCase()); // ANNE

Los métodos length, includes(), toLowerCase() y toUpperCase() son solo una pequeña parte de lo que el tipo String puede hacer en JavaScript. De hecho, existen decenas de métodos integrados que permiten:

  • recortar cadenas (slice, substring)
  • dividir cadenas en partes (split),
  • reemplazar partes de una cadena (replace),
  • eliminar espacios innecesarios (trim),
  • encontrar posiciones de caracteres (indexOf, lastIndexOf)

Estas herramientas hacen que trabajar con texto sea cómodo, flexible y potente, especialmente en el desarrollo web, donde las cadenas de texto están por todas partes: en formularios, títulos, mensajes, URLs y datos.
